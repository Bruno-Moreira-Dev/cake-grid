My Cake App

Little John was a boy that loved cake. He loved it so much that he decided to create an app just to catalog his caking experiences.

Your requisites are:

  • 1. Display a grid view catalog of cakes reading the file cakes.json [cakelist]
  • 2. Display a detailed page for the selected cake.

Bonus objectives:

  • 3. Create the functionality for adding or removing a cake. (You can add comments to your code to simulate making HTTP requests to an API to save it on the server)
  • 4. Optimize your components so they don't make unecessary requests to the Cake API (Reading the cake file should be interpreted as making a GET request to an API)

Optional objective:

  • 5. The app should be optmized to work with the file cakes-10000.json [10000 Cakes]

IMPORTANT: the front-end of your solution must have a nice appearance, don't think only as a programmer. It should also be responsive.
